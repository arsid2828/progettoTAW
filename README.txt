ISTRUZIONI PER FAR PARTIRE IL SITO SkyJourney

1) Scaricare Docker e tenerlo aperto in background
2) Assumendo che il progetto sia dentro la cartella progettoTAW, entrare nella cartella con cd Desktop\progettoTAW
3) Eseguire docker compose up --build -d 
4) Per avviare il sito andare al link http://localhost:8080/
5) Per chiudere il sito e Docker, eseguire docker compose down
